Criando nav's de level 2

Podemos adicionar dentro de um link principal uma outra lista de opções, criando uma espécie de segundo nível.

Link 1 Link 1 Link 2 Link 3 Link 2 Link 3 
                    
                        <menu class="c-nav--gary" role="navigation">
                            <menu class="c-nav__menu--first" role="menubar">
                                <menuitem class="c-nav__item--first" role="menuitem"><a class="has-dropdown" href="#" onclick="NavController.dropdown(this, event);" role="link">Link 1</a>
                                    <menu class="c-nav__menu--second is-hide" role="menubar">
